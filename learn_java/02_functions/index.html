Functions

In Java there's no such thing as a pure function -- every function is attached to a Class. But "public static" functions are very similar to pure functions so we'll be building some of those here.

In FunctionsTest.java you'll see tests for:

  • hello - returns a string
  • add - adds two numbers
  • ftoc - converts fahrenheit to celcius
  • makeArray - converts its arguments into an array
  • sum - adds all the numbers inside an array together
