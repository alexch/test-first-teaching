Introduction to Test-First Training - IDEA Instructions

First, install and configure IDEA, according to the instructions on the main page
  1. Open 01_intro/IntroTest.java
  2. Right-click the tab of the source code window.
  3. Select "Run IntroTest"
    If "Run IntroTest" is not available, make sure that 01_intro is marked as a Test Sources Root by right-clicking on it and selecting "Mark Directory as... > Test Sources Root" or setting it in the Project Structure settings.
  4. See how the first test passes? It's green. GREEN IS GOOD!
  5. See how the second test fails? That's okay... now you're going to fix it. When you look at the test results you will see an error message that looks something like this: 
    
            junit.framework.ComparisonFailure:
            Expected :Goodbye
            Actual   :Farewell
            ...
            at IntroTest.testGoodbye(IntroTest.java:14)
  6. Open the file Intro.java and edit the code inside getGoodbye to make it pass.
  7. Now create an entirely new test containing this code: 
    
                public void testHowAreYou()
                {
                    Intro x = new Intro();
                    assertEquals("Fine", x.sayHowAreYou());
                }
  8. Run the test and watch it fail.
  9. Write some code inside Intro.java.
  10. Run the test and watch it pass!
  11. Give yourself a high five. You deserve it!
