Test-First Training: Java

  1. Clone this repo from GitHub
  2. Download and install a JDK (Java Development Kit)
  3. Download and Launch JetBrains IntelliJ IDEA
  4. Open this project inside test-first-teaching/learn_java
  5. Open intro/index.html in a local browser
  6. Open IntroTest.java in IntellJ
  7. Set up your JDK
    1. Click "Setup JDK"
    2. Click "Configure..."
    3. Click the "+" button
    4. In "Add New SDK" menu, click "JDK"
    5. Click "OK" a few more times
  8. Configure the compiler output directory
    1. Select "File > Project Structure"
    2. Click the three dots beneath "Project Compiler Output"
    3. Click the brown folder icon to select the project directory
    4. Click the "Classes" directory
  9. Configure Modules as Tests
    1. Select "File > Project Structure"
    2. Click "Modules"
    3. Select all directories except "Classes" and ".idea"
    4. Click "Mark as: Tests"
  10. Add junit.jar to the classpath
    1. Select "File > Project Structure"
    2. Select "Add Library"
    3. Select the junit-4.12.jar file in this dir
